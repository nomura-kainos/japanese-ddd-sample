# 開発環境の構築方法

開発環境の最初の構築は以下の手順で行う。

## Docker環境をbuild

```bash
$ make install
```
※動作未検証

環境構築完了の確認は以下のURLで行う。

`http://localhost:10080`
または
`http://192.168.99.100:10080/`

<br>

# 開発環境の構築方法
開発環境の起動・停止は以下の手順で行う。
<br>
※PCをシャットダウンする前に必ず停止すること

## Docker環境を起動

```bash
$ make up
```

## Docker環境を停止

```bash
$ make up
```
